Java Edition — оригінальна версія Minecraft розроблена компанією Mojang Studios для Windows, macOS, та Linux. Нотч розпочав розробку 10 травня 2009 року, публічно оприлюднивши Minecraft 16 травня 2009 року. Повний випуск гри відбувся 18 листопада 2011, на MineCon 2011.Mojang здійснює продаж Java Edition безпосередньо, тому сертифікація власників платформи не потрібна. Розробники можуть додавати функції, недоступні в інших виданнях. Java Edition має свій лаунчер. Для гри потрібний обліковий запис Mojang. Через лаунчер доступні всі версії і попередні збірки. Лаунчер дозволяє створити окремі профілі, корисні при використанні модифікацій, снапшотів, і старших версій. Лише видання Java має офіційне програмне забезпечення для гравців для створення власних серверів. Код Java Edition легше змінювати, в порівнянні з іншими виданнями, що робить її на сьогоднішній день найбезпечнішим майданчиком для створення модів та користувацьких серверів. Realms — окрема послуга, відмінна від Realms з Bedrock Edition. На відміну від Bedrock Edition, який недоступний на комп'ютерах без Windows 10, Java Edition підтримує різні операційні системи. Всі власники Java Edition можуть купити безкоштовну копію Bedrock Edition для Windows 10. Оскільки вартість видань однакова, придбати Java Edition краще, ніж купувати Windows 10 Edition, так як остання не дозволить придбати Java Edition безкоштовно.
